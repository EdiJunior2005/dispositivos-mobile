**Estudantes:**

## Objetivo da Atividade
O objetivo desta atividade é demonstrar a correta configuração do ambiente de desenvolvimento para a linguagem Dart. O projeto consiste na criação e execução de um programa simples que imprime a mensagem "Hello World" no terminal, visando também a prática de versionamento de código utilizando Git e publicação no GitHub.

## Instruções para Executar o Programa
Para executar o programa em sua máquina, é necessário ter o [Dart SDK](https://dart.dev/get-dart) instalado e configurado corretamente.
E execute o camando: dart run main.dart

A saída deverá ser: Hello World
